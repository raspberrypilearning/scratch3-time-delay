يمكنك تشغيل البرنامج النصي بعد عدة ثوانٍ من النقر فوق العلم الأخضر:

```blocks3
when flag clicked
wait [5] seconds // change the delay
```

Add blocks underneath the `wait`{:class="block3control"} block to run after the delay.
